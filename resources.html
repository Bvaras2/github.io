Sto Rox Code Club Member Resources

Bitsbox- Monthly subscription, online IDE using subset of Javascript to code apps using text based coding. Books with activities are available in Digilab, check it out: Bitsbox

Code.org-Fun, interactive tutorials on HTML, JavaScript and other Computer Science topics, Leading organization in U.S. advocating for quality Computer Science in all schools, ask for a log-in to our class. Check it out: Code.org

Pencilcode.net, is an interactive sandbox for learning to program by drawing art, playing music and creating games. Programs preload the pencilcode library to us turtle graphics functions. Learn CSS, HTML, JavaScript and/or CoffeeScript. Check it out: PencilCode

Earsketch-is a DAW (Digital Audio Workstation) that lets users create music using Python or JavaScript. Has built in sound clips made by professionals in the music industry. Earsketch is a project for Georgia Tech. Check it out: Earsketch

OSSU-Open Source Society University-Path to a free self-taught education in Computer Science. Check it out: OSSU-Computer Science
